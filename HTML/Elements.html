HTML ELEMENTS

In HTML, There are opening tags and closing tags MDN Docs: ElementsOpening:
<tag>
Closing:
</tag>

Element: Is an opening tag, a closing tag, and the code in between.

*Element note:

Most have to be closed to function, if they don't they are considered an "empty tag", meaning they have no closing tag.

MDN Docs: Elements

There are different element types:


<a>:
Anchor Tag is used to create links. (ex: <a> href="insert link here"</a>) 
<br>:
Line Break - Kinda like hitting enter. 
<em>:
EM needs to be closed, makes text italic. 
<hr>:
Horizonal Rule -"Draws" a line across the site. 
<stong>:
Strong needs to be closed, makes the text bold. 
<img>:
Img tag allows us to add images. src stands for source. Alt is used if the image cant be shown, it displays placeholder text instead (ex: <img> src="insert link here" alt="picture description goes here")
